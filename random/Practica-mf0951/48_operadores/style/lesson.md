* Operadores asignación:

=	x = y	x = y
+=	x += y	x = x + y  //Asigna añadiendo, le añade el valor de y

*=	x *= y	x = x * y
/=	x /= y	x = x / y
%=	x %= y	x = x % y //
**=	x **= y	x = x ** y
