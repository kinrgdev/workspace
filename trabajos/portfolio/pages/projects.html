Pizza World

Este proyecto es una web ficticia de una pizzería, desarrollada como parte del proyecto final de un curso de Udemy. Durante su desarrollo, consolidé mis conocimientos en HTML y CSS, además de descubrir las posibilidades de JavaScript para mejorar la interacción del usuario.

Características del proyecto: Diseño moderno y responsive: La web se adapta a distintos dispositivos, garantizando una experiencia fluida tanto en computadoras como en móviles. Menú interactivo: Presenta una lista de pizzas con imágenes atractivas en formato carousel. Animaciones y efectos visuales: Implementados con CSS y JavaScript para mejorar la estética y la experiencia del usuario. Formulario de contacto y Navegación intuitiva.

SW Hyperspace Jump

Este proyecto es una animación en Canvas que simula el icónico viaje hiperespacial de Star Wars.
Fue desarrollado como parte de mi aprendizaje en JavaScript, explorando la manipulación del lienzo y la generación de efectos visuales dinámicos.
Efecto inmersivo: Simula estrellas moviéndose a gran velocidad hacia el espectador, recreando la sensación de viajar a la velocidad de la luz. La animación se genera mediante JavaScript y la API de Canvas, permitiendo un control detallado sobre los elementos gráficos. Cálculos y parámetros para modificar, estrellas, velocidad y profundidad.

Shenron

En la escena, las legendarias bolas de dragón giran alrededor de Shenron, el mítico dragón de la serie Dragon Ball, describiendo una órbita elíptica suave y continua sobre un fondo de cielo negro con tormenta eléctrica.
Al situar el cursor sobre Shenron, aparece su frase característica, invitando a pedir tus deseos, lo que añade interactividad y dinamismo a la experiencia. Además, el usuario puede seleccionar entre tres idiomas —catalán, español e inglés— para adaptar el contenido al público.

Este trabajo fue realizado como parte del curso CTS con motivo del Día de Sant Jordi, combinando animación con HTML, CSS y JavaScript para demostrar habilidades en animación avanzada, eventos de usuario y localización multilingüe.

Croqueta panda

Proyecto creativo para una página de error 404, en el que un oso panda animado entra y sale de pantalla mientras hace la croqueta de forma continua y en bucle. La imagen del panda es un GIF generado mediante inteligencia artificial, y la animación que simula el movimiento, rebote y recorrido fue implementada completamente con JavaScript.

Esta propuesta aporta un toque de humor y personalidad a una página normalmente estática y aburrida, mostrando habilidades en animación programada, manipulación del DOM y uso de recursos generados por IA para enriquecer la experiencia del usuario.
